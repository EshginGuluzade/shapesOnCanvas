This app creates simple rectangles and squares on canvas.

1. You need to enter width, height, red color value for RGB, green color value for RGB, blue color value
   for RGB of the canvas. Color values must be between 0 and 255.

2. Then programs asks the x and y coordinates of the shape which will be drawn on the canvas. After that
   you need to enter the parameters of the shape.

3. If you want to quit the program then write 'quit' and press enter.
